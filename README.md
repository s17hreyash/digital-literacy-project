## 📌 Project Overview
This project is created as part of the Digital Literacy course. It focuses on building awareness about digital tools, online safety, professional communication, and cybercrime prevention.

The project demonstrates practical understanding through multiple tasks including infographic design, digital portfolio creation, platform exploration, email etiquette, and cybercrime awareness.
